# Target Brazil E-Commerce Analytics

## Problem Statement

Analyze Target Brazil's e-commerce operations by evaluating customer
behavior, sales trends, payment patterns, and logistics performance
to derive actionable insights that improve business growth,
operational efficiency, and customer satisfaction.

## Tech Stack

- SQL (MySQL)
- Python
- Power BI
- Excel

## Dataset

- 100k+ orders
- 8 relational tables

## Key Analyses

1. Customer Analysis
2. Sales Trends
3. Late Delivery Analysis
4. Freight Cost Analysis
5. Payment Behavior Analysis
6. Seasonal Demand Analysis
7. Customer Retention Analysis

## Key Findings

- 8% of deliveries were delayed.
- SP contributes the highest customer base.
- Significant freight variations exist across states.
- Credit cards are the most preferred payment method.
- November exhibits peak demand.

## Business Recommendations

- Optimize logistics in high-delay regions.
- Improve freight efficiency.
- Introduce targeted marketing campaigns.
- Enhance customer retention strategies.
